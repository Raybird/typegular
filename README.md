# typegular
learn typescript angular

## require

  * php 5.4 + for simple built in server.
  * nodejs and npm.

## usage

  1. clone project.
  2. npm install.
  3. npm run build.
  4. npm run start.
  5. open browser url: [http://localhost:8080](http://localhost:8080)
